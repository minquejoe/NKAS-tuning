# NIKKEAutoScript

『勝利の女神：NIKKE』の自動デイリースクリプトです。すべての PC／エミュレーター クライアントをサポートしています（**中国本土版を除く**）。  
このプロジェクトは [NIKKEAutoScript](https://github.com/takagisanmie/NIKKEAutoScript) からフォークされたものです。

**このプロジェクトが役に立ったと思ったら、ぜひ ⭐Star を付けて応援してください！**

### 謝辞

**Alas** のすべての開発者の皆さんに感謝します。今でもあのソースコードは本当にすごいと思っています。  
「Pythonって、こんな書き方できるのか！？」と学ばせてもらいました。  
そして自分の最初のバージョンを見返して、「あれ、ただのクソコードじゃん」と思いました😅😅😅  
今の NKAS も正直クソですが、少なくとも動きます😅😅😅
